# vrchat-world-template
The minimal empty world of VRChat

## Versions
- Unity 2019.4.31f1 
- VRChat3 2022.06.03.00.03 
- UdonSharp v0.20.3

## Structure
```markdown
├ Assets/
  ├ Gizmos
  ├ Scenes 
  ├ Main  `Your custom assets`
  ├ SerializedUdonPrograms 
  ├ Udon
  ├ UdonSharp
  ├ VRCSDK
  ├ VRChat Examples
├ Packages/
├ ProjectSettings/ 
├ .gitignore 
├ LICENSE 
├ README.md 
```
